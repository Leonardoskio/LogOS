# Lista Autisti

Questa pagina raccoglie gli autisti presenti nel vault operativo.

## Vista prevista

La lista dovrebbe mostrare:

```text
nome e cognome
stato
scadenza patente
ultima posizione nota
spedizioni collegate
```

## Regola

La posizione live non deve essere aggiornata continuamente nel vault. Il backend
gestira lo stato corrente e potra salvare snapshot quando serve.
