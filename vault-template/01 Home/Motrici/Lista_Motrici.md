# Lista Motrici

Questa pagina raccoglie le motrici presenti nel vault operativo.

## Vista prevista

La lista dovrebbe mostrare:

```text
targa
marca
stato
scadenza assicurazione
chilometri percorsi
spedizioni collegate
```

## Regola

La motrice puo avere uno stato operativo corrente, ma lo storico dei viaggi si
ricava dalle spedizioni in `02 Archivio` tramite `motrice_id`.
