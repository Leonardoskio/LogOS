# Lista Clienti

Questa pagina raccoglie i clienti presenti nel vault operativo.

## Vista prevista

Quando il backend o un sistema di query Obsidian sara configurato, questa vista
dovra mostrare:

```text
nome cliente
riferimento
sede
numero spedizioni derivate da 02 Archivio
ultimo viaggio
```

## Regola

Il numero di spedizioni non va aggiornato a mano nella nota cliente. Deve essere
calcolato contando le spedizioni con lo stesso `cliente_id`.
