# Dashboard AI

Questa pagina e il punto di ingresso per report, analisi e suggerimenti generati
o assistiti dall'AI.

## Fonti

L'AI deve leggere dati strutturati da:

```text
02 Archivio/
01 Home/Clienti/
01 Home/Autisti/
01 Home/Motrici/
```

## Output previsti

```text
report giornalieri
analisi clienti
analisi utilizzo motrici
segnalazioni dati mancanti
suggerimenti logistici
```

## Regola

I report AI devono citare gli ID delle spedizioni analizzate. Se un dato manca,
il report deve segnalarlo invece di inventarlo.
